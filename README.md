# Using LLMs for Knowledge Component-level Correctness Labeling in Open-ended Coding Problems
This repository contains the code for the paper <a href="https://arxiv.org/abs/2602.17542">Using LLMs for Knowledge Component-level Correctness Labeling in Open-ended Coding Problems</a>. To our knowledge, this work is the first work to explore the use of LLMs for KC-level correctness labeling in student-written code. It shows better fit to the Power law of Practice than using problem-level correctness for student modeling.
